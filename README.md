### Build the docker image:
make build
### Run the container:
make run
### Exec into the container:
make exec
### Go to the notebook:
cd /app/app
### Run the notebook:
jupyter-notebook --ip=0.0.0.0 --port=8000 --allow-root
### Access the notebook
In your browser, go to 0.0.0.0:8000
Open the notebook
Have fun!
